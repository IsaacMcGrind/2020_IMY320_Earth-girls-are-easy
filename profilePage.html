Project Tree

The last Project Management tool you'll ever need

Name : Project Tree

Initialized: 2020

Creators: The B-Team

History:

A project started by the The B-Team, commissioned by EpiUse for a software engineering module COS301 at the university of Pretoria. Project Tree is s project management system specifically deisnged for large corparations to create Projects and tasks with certain dependencies and view those tasks in a graphical form for easier viewing.
