ObjectID

IDing your your car should be as easy as breathing

Name : ObjectID

Initialized: 2020

Creators: Ctrl + Intelligence

History:

ObjectID is a system which aims to analyse and store information on vehicles that have been identified by CCTV camera's, it is then able to identify vehicles based on what it has learnt from these images. What sets it apart from the of the shelf offering is the ability to recognise vehicles based on their damage or distinguishing features. As a user you can login into the system, and once logged into the system you can search to find vehicles matching some criteria.

Project Tree

The last Project Management tool you'll ever need

Name : Project Tree

Initialized: 2020

Creators: The B-Team

History:

A project started by the The B-Team, commissioned by EpiUse for a software engineering module COS301 at the university of Pretoria. Project Tree is s project management system specifically deisnged for large corparations to create Projects and tasks with certain dependencies and view those tasks in a graphical form for easier viewing.
